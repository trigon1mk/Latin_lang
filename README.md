
# Latin Translator

Цей проект є простим веб-додатком для перекладу слів з латини на українську мову.

## Файли проекту
- `index.html`: Головна HTML-сторінка.
- `style.css`: Стилі для сторінки.
- `translator.js`: Скрипт для перекладу тексту.

## Як використовувати
1. Введіть текст латинською мовою у текстове поле.
2. Натисніть кнопку "Перекласти".
3. Переклад з'явиться під текстовим полем.

## Налаштування GitHub Pages
Ви можете запустити цей проект за допомогою GitHub Pages:
1. Перейдіть у налаштування репозиторію.
2. У секції **Pages** оберіть гілку `main` та папку `root`.
3. Збережіть налаштування.
